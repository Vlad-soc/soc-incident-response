# SOC Incident Response Practice

This repository contains hands-on practice cases related to Security Operations Center (SOC) activities.

## Focus Areas
- Alert triage
- Log analysis (Windows / Linux)
- Incident investigation
- Incident documentation and reporting

## Tools & Concepts
- SIEM fundamentals
- Windows Event Logs
- Linux authentication logs
- Network basics (IP, DNS, HTTP)

## Purpose
The goal of this repository is to demonstrate practical understanding of SOC workflows and incident response fundamentals.

All cases are based on lab simulations and training environments.
